# hotyogapower.com

The public **storefront / front door** for HYP Yoga Рђћ its own static site, deliberately
separate from the `hyp.yoga` web app and backend.

> **Hard rule:** Anything built on `hyp.yoga` stays unchanged. This site is 100% additive.
> It is static, runs **zero JavaScript**, collects **no data**, and only *links* to hyp.yoga.

## Pages

| Path | File | Purpose |
|------|------|---------|
| `/` | `index.html` | Front door Рђћ brand hero + two doors (the app ┬и the web) |
| `/app` | `app.html` | The iOS app Рђћ overview + "coming soon to the App Store" + tier ladder |
| `/web` | `web.html` | Gateway Рђћ Sign up Рєњ `hyp.yoga/signup`, Log in Рєњ `hyp.yoga/login` |

Design language is inherited from hyp.yoga (cream + warm gold, Cormorant Garamond + Inter,
self-hosted for GDPR). Shared styles live in `style.css`; fonts in `fonts/` (Cormorant + Inter
subset only). The OG image is a placeholder generated by `scripts/make-og.py` (Georgia stand-in
for Cormorant) Рђћ replace `images/og-image.png` with a designed asset when ready.

## Deploy (one-time setup)

This repo is a normal clone Рђћ **no** `index.lock` / `/tmp/hyp-deploy` workaround (that's
hyp-yoga-only). Just push; Netlify auto-deploys.

1. **GitHub repo**
   ```bash
   gh repo create joshuajudd-usa/hotyogapower --public \
     --source=/Users/joshua/Code/hotyogapower --remote=origin --push
   ```
   (or create it on github.com and `git remote add origin Рђд && git push -u origin main`)

2. **Netlify site** Рђћ New site Рєњ *Import from Git* Рєњ pick `hotyogapower`.
   Build command: *(none)* ┬и Publish directory: `.` ┬и Functions: *(none)*.

3. **Domain** Рђћ Netlify Рєњ Domain settings Рєњ add `hotyogapower.com` **and** `www.hotyogapower.com`.
   Set the **apex (`hotyogapower.com`) as Primary** so `www` auto-301s to it.
   Point DNS at the registrar (Netlify nameservers, or the A/ALIAS + CNAME records Netlify shows).
   Let Netlify provision the Let's Encrypt cert.

4. **Verify** Рђћ visit `https://hotyogapower.com`; confirm the URL sticks (no redirect to hyp.yoga),
   both doors work, and `/web` lands on hyp.yoga signup/login.

Thereafter: edit Рєњ `git commit` Рєњ `git push` Рєњ Netlify rebuilds.
